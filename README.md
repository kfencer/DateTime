KFencer DateTime
=================

Date & time library.

Requirements
------------

- PHP 7.4+

Installation
------------

Install the bundle using composer:
Add to composer.json
```json
{
    "repositories": [
        {
            "type": "git",
            "url": "https://github.com/kfencer/DateTime"
        }
    ]
}
```

```sh
composer require kfencer/date-time
```
